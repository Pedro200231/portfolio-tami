# 🎯 Portfolio Tami Uemura - Analista de QA

Um site pessoal moderno, elegante e responsivo criado especialmente para **Tami Uemura**, analista de QA especializada em qualidade de software e experiência do usuário.

## ✨ Características

- **Design moderno e elegante** com foco em UX/UI
- **Totalmente responsivo** (desktop, tablet, mobile)
- **Modo claro/escuro** com alternância suave
- **Animações fluidas** e interações cativantes
- **Performance otimizada** com lazy loading e throttling
- **Acessibilidade completa** (WCAG 2.1)
- **SEO otimizado** com meta tags adequadas
- **Código limpo** e bem documentado

## 🛠️ Tecnologias Utilizadas

- **HTML5** semântico e acessível
- **CSS3** com variáveis CSS e grid/flexbox
- **JavaScript ES6+** modular e orientado a objetos
- **Sem frameworks externos** - código puro e performático

## 🚀 Como Executar

1. **Clone ou baixe** os arquivos
2. **Abra** o `index.html` em qualquer navegador moderno
3. **Para desenvolvimento local**, use um servidor HTTP simples:

   ```bash
   # Com Python
   python -m http.server 8000
   
   # Com Node.js (npx)
   npx serve .
   
   # Com Live Server (VS Code)
   Extensão Live Server
   ```

## 📱 Responsividade

O site foi desenvolvido com abordagem **mobile-first** e é totalmente responsivo:

- **Desktop** (1200px+): Layout completo com grid de 2 colunas
- **Tablet** (768px-1199px): Layout adaptado com elementos reorganizados
- **Mobile** (até 767px): Layout vertical com menu hamburger

- **Secundária:** Lilás claro (#D6C8F4)

## ⚡ Performance

- **Animações otimizadas** com `transform` e `opacity`
- **Lazy loading** para imagens
- **Throttling** para eventos de scroll
- **Debouncing** para redimensionamento
- **Intersection Observer** para animações
- **CSS Grid** e **Flexbox** para layouts eficientes

## ♿ Acessibilidade

- **Navegação por teclado** completa
- **Screen readers** suportados com ARIA
- **Contraste adequado** (WCAG AA)
- **Foco visível** em todos elementos interativos
- **Texto alternativo** para imagens e ícones
- **Semântica HTML** correta

## 🔧 Funcionalidades JavaScript

### Managers Implementados

- **ThemeManager**: Alternância entre modo claro/escuro
- **NavigationManager**: Menu responsivo e navegação suave
- **ScrollAnimations**: Animações baseadas em scroll
- **SkillCardsManager**: Interações nos cards de habilidades
- **ContactFormManager**: Validação e envio do formulário
- **PerformanceManager**: Otimizações de performance
- **AccessibilityManager**: Melhorias de acessibilidade

## 📧 Configuração do Formulário

O formulário atualmente simula o envio. Para implementar envio real:

1. **Backend próprio**: Configure um endpoint para receber os dados
2. **Serviços como Formspree**: Integre com serviços de formulário
3. **Netlify Forms**: Se hospedado no Netlify, use os forms nativos

## 🌐 Deploy

O site pode ser facilmente hospedado em:

- **GitHub Pages**: Commit os arquivos e ative Pages
- **Netlify**: Drag & drop da pasta ou conecte o repositório  
- **Vercel**: Import do repositório GitHub
- **Surge.sh**: CLI simples para deploy estático
